## Nicht verwendete Bibliothekselemente

Diese Elemente sind in der IMR UI Library bzw. Angular Material vorhanden, werden aber in **keinem** App-Modul direkt eingesetzt.

### IMR UI Library – nicht verwendete Elemente

| Element | Beschreibung |
|---------|-------------|
| `imr-autocomplete` | Autocomplete-Wrapper |
| `imr-card-content` | Karteninhalts-Container |
| `imr-chips` | Chips-Input-Feld (distinct von `imr-chip-set`) |
| `imr-expansion-panel-header` | Expansion-Panel-Header-Wrapper |
| `imr-form-actions` | Formular-Aktionsleiste |
| `imr-input` | Input-Feld-Wrapper |
| `imr-label` | Label-Wrapper |
| `imr-section` | Abschnitts-Container |
| `imr-step` | Einzel-Schritt im Stepper |
| `imr-stepper` | Stepper-Container |
| `imr-toolbar` | Toolbar-Wrapper |
| `ui-control-errors` | Fehler-Direktive für Formular-Controls |

### Angular Material – nicht direkt verwendete Elemente

Diese mat-Elemente existieren und werden indirekt über ihre `imr-*`-Wrapper eingesetzt, kommen aber **nirgendwo als direkte `mat-*` Tags** in den App-Templates vor:
